# Zagara

## 简介
#### 这是一个基于Unity的软渲染器

![image](https://github.com/justalittlefat/Zagara/blob/master/Images/Config.png)
* 配置输出目录
* 点击"Fire!"按钮，即可渲染出一帧图像
* 全部渲染过程由CPU处理，可以在内存中追踪所有数据
